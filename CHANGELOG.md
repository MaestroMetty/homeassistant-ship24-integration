## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Ship24 API integration with two-layer architecture
- Config flow for API key setup
- Sensor entities for each tracked package
- Webhook support for real-time updates
- Polling fallback mechanism
- Service calls for adding/removing packages
- Support for custom package names
- Complete tracking event timeline
- Location data extraction
- Status mapping from Ship24 to standardized format

### Features
- Track multiple packages simultaneously
- Real-time updates via webhooks
- Detailed tracking information
- Custom names for packages
- Automatic status updates
