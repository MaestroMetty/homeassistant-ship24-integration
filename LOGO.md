# Logo

## Logo Requirements

The integration logo should be placed in the repository root as `logo.png`.

### Specifications

- **Format**: PNG
- **Size**: 128x128 pixels (recommended, minimum 100x100)
- **Background**: Transparent or white
- **File name**: `logo.png` (must be in repository root)

### Logo Location

```
homeassistant-ship24-integration/
├── logo.png          ← Place logo here
├── hacs.json
├── README.md
└── custom_components/
    └── ship24/
```

### Where Logo is Used

The logo will be automatically displayed in:
- HACS integration browser
- Integration settings page in Home Assistant
- Integration card in HACS

### Creating a Logo

You can create a logo using:
- Ship24 branding guidelines (if available)
- A package/tracking icon
- Your custom design

The logo should represent package tracking or the Ship24 service.
